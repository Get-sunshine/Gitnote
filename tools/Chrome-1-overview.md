# 总览

## 打开 DevTools

* check DOM or CSS , Mac:  Command+Option+C  Win: Control+Shift+C 。或者，在页面上右键，选择 inspect （检查、审查）
* run JavaScript or log, Mac: Command+Option+C Win: Control+Shift+C。打开 console。
